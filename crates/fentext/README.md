# FenText: Text-Based Game Development with Swamp

FenText is a specialized engine for purely text-based games (e.g, roguelikes, snake). It combines the crisp terminal UI capabilities of `fentext_ui` with the robust, type-safe embedded scripting power of **Swamp**.
