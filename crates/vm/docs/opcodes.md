# Swamp VM Opcode Documentation

The opcodes for the Swamp Virtual Machine. The VM features a minimized 32 bit
ARM (RISC)-inspired architecture with 256 general-purpose registers (only 0-127 is used for now). Includes complex instructions as "Runtime Functions".

| Mnemonic                          | U8 Value | Operands                                 | Description                                                                                                                                                                       |
| --------------------------------- | -------- | ---------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Control Flow**                  |          |                                          |                                                                                                                                                                                   |
| `nop`                             | 0        |                                          | No operation.                                                                                                                                                                     |
| `hlt`                             | 1        |                                          | Halts VM execution and relinquishes control back to the host. Used as a "return" for functions called by the host.                                                                |
| `panic`                           | 2        | `Rn`                                     | Initiates a panic state in the VM. $Rn$ points to a string header with the explanation for the panic.                                                                             |
| `brk`                             | 3        |                                          | Software breakpoint. _not implemented yet_.                                                                                                                                       |
| `b.ne`                            | 25       | `#branch_target`                         | Branches if Z flag is not set (Not Equal). `PC += #branch_target`.                                                                                                                |
| `b.eq`                            | 26       | `#branch_target`                         | Branches if Z flag is set (Equal). Branch target is `PC += #branch_target`.                                                                                                       |
| `b`                               | 27       | `#branch_absolute`                       | Unconditional branch. Sets the `PC=#branch_absolute`.                                                                                                                             |
| `call`                            | 28       | `#branch_target`                         | Calls a subroutine at address #branch_absolute, but first it is pushing current `PC` and Frame Pointer (FP).                                                                      |
| `ret`                             | 30       |                                          | Returns from a subroutine, restoring `PC` and frame pointer (FP).                                                                                                                 |
| **Frame and Function Management** |          |                                          |                                                                                                                                                                                   |
| `enter`                           | 29       | `#frame_size`                            | Sets up a new stack frame of size `#frame_size`. `SP` is changed by `#frame_size`                                                                                                 |
| `ldfp`                            | 38       | `Rd, #fp_offset, #len`                   | Loads a value from the pointer at FP + `#fp_offset` into register $Rd$.                                                                                                           |
| `stfp`                            | 41       | `Rn, #fp_offset, #len`                   | Stores the value from register $Rn$ to the current stack frame at `#fp_offset`.                                                                                                   |
| **Integer Arithmetic**            |          |                                          |                                                                                                                                                                                   |
| `add`                             | 4        | `Rd, Rn, Rm`                             | Adds unsigned 32-bit integers. $Rd = Rn + Rm$.                                                                                                                                    |
| `mul`                             | 5        | `Rd, Rn, Rm`                             | Multiplies unsigned 32-bit integers. $Rd = Rn * Rm$.                                                                                                                              |
| `sub`                             | 6        | `Rd, Rn, Rm`                             | Subtracts unsigned 32-bit integers. $Rd = Rn - Rm$.                                                                                                                               |
| `sneg`                            | 7        | `Rd, Rm`                                 | Negates a signed 32-bit integer. $Rd = -Rm$.                                                                                                                                      |
| `smod`                            | 8        | `Rd, Rn, Rm`                             | Signed 32-bit integer modulo. $Rd = Rn \% Rm$.                                                                                                                                    |
| `sdiv`                            | 9        | `Rd, Rn, Rm`                             | Signed 32-bit integer division. $Rd = Rn / Rm$.                                                                                                                                   |
| **Float Arithmetic**              |          |                                          |                                                                                                                                                                                   |
| `fmul`                            | 10       | `Rd, Rn, Rm`                             | Multiplies 32-bit fixed point numbers. $Rd = Rn * Rm$.                                                                                                                            |
| `fmod`                            | 11       | `Rd, Rn, Rm`                             | 32-bit fixed point modulo. $Rd = Rn \% Rm$.                                                                                                                                       |
| `fdiv`                            | 12       | `Rd, Rn, Rm`                             | Divides 32-bit fixed point numbers. $Rd = Rn / Rm$.                                                                                                                               |
| **Comparisons and Flags**         |          |                                          |                                                                                                                                                                                   |
| `lt`                              | 13       | `Rn, Rm`                                 | Sets Z flag if signed 32-bit $Rn < Rm$.                                                                                                                                           |
| `le`                              | 14       | `Rn, Rm`                                 | Sets Z flag if signed 32-bit $Rn \le Rm$.                                                                                                                                         |
| `gt`                              | 15       | `Rn, Rm`                                 | Sets Z flag if signed 32-bit $Rn > Rm$.                                                                                                                                           |
| `ge`                              | 16       | `Rn, Rm`                                 | Sets Z flag if signed 32-bit $Rn \ge Rm$.                                                                                                                                         |
| `eq8`                             | 17       | `Rn, #imm8`                              | Sets Z flag if the lowest 8 bits of $Rn$ equal immediate value `#imm8`.                                                                                                           |
| `cmp`                             | 18       | `Rn, Rm`                                 | Compares two registers $Rn$ and $Rm$ and sets Z flag depending on equality.                                                                                                       |
| `cmpblk`                          | 19       | `Rn, Rm, #len`                           | Compares blocks of memory pointed to by $Rn$ and $Rm$ with length `#len`. Sets flags (i.e. Z).                                                                                    |
| `tst`                             | 20       | `Rn`                                     | Tests the value of register $Rn$ and updates the Z flag.                                                                                                                          |
| `notz`                            | 22       |                                          | Inverts the state of the Z flag. **TODO: should be removed**.                                                                                                                     |
| `movz`                            | 23       | `Rd`                                     | Stores the state of the Z flag into register $Rd$.                                                                                                                                |
| `stnz`                            | 24       | `Rd`                                     | Stores the inverse state of the Z flag into register $Rd$. **TODO: should be removed**.                                                                                           |
| **Move and Copy**                 |          |                                          |                                                                                                                                                                                   |
| `mov`                             | 31       | `Rd, Rm`                                 | Moves the value from register $Rm$ to register $Rd$. $Rd = Rm$.                                                                                                                   |
| `movb`                            | 34       | `Rd, #imm8`                              | Moves an 8-bit immediate value `#imm8` into the lower 8 bits of register $Rd$.                                                                                                    |
| `movw`                            | 35       | `Rd, #imm32`                             | Moves a 32-bit immediate value `#imm32` into register $Rd$.                                                                                                                       |
| `blockcopy`                       | 32       | `Rd, Rn, #len`                           | Copies a block of memory of length `#len` from address $Rn$ to address $Rd$. TODO: Maybe should be called `memmov`?.                                                              |
| **Load Operations**               |          |                                          |                                                                                                                                                                                   |
| `lea`                             | 33       | `Rd, #offset`                            | Loads the effective address calculated from `FP` + `#offset` into register $Rd$.                                                                                                  |
| `ldb`                             | 36       | `Rd, [Rn, #offset]`                      | Loads a 8-bit value from memory at address $Rn$ + `#offset` into register $Rd$.                                                                                                   |
| `ldb`                             | ?        | `Rd, #absolute_addr`                     | Loads a 8-bit value from the absolute memory address `#absolute_addr` into register $Rd$. Used mainly for constants placed at known locations.                                    |
| `lduh`                            | ?        | `Rd, [Rn, #offset]`                      | Loads an unsigned 16-bit value from memory at address $Rn$ + `#offset` into register $Rd$.                                                                                        |
| `lduh`                            | ?        | `Rd, #absolute_addr`                     | Loads an unsigned 16-bit from the absolute memory address `#absolute_addr` into register $Rd$. Used mainly for constants placed at known locations.                               |
| `ldw`                             | 37       | `Rd, [Rn, #offset]`                      | Loads a 32-bit value from memory at address $Rn$ + `#offset` into register $Rd$.                                                                                                  |
| `ldw`                             | ?        | `Rd, #absolute_addr`                     | Loads a 32-bit value from the absolute memory address `#absolute_addr` into register $Rd$. Used mainly for constants placed at known locations.                                   |
| **Store Operations**              |          |                                          |                                                                                                                                                                                   |
| `stb`                             | 40       | `Rn, [Rd, #offset]`                      | Stores a 8-bit value from register $Rn$ to memory at address $Rd$ + `#offset`.                                                                                                    |
| `sth`                             | ?        | `Rn, [Rd, #offset]`                      | Stores a 16-bit value from register $Rn$ to memory at address $Rd$ + `#offset`.                                                                                                   |
| `stw`                             | 39       | `Rn, [Rd, #offset]`                      | Stores a 32-bit value from register $Rn$ to memory at address $Rd$ + `#offset`.                                                                                                   |
| **Memory Management**             |          |                                          |                                                                                                                                                                                   |
| `alloc`                           | 42       | `Rd, #size`                              | Allocates a block of heap memory of size `#size` and places the absolute address in register $Rd$.                                                                                |
| `memclrf`                         | 21       | `#offset, #size`                         | Clears a block of memory on the current frame offset of size `#size`. Used mainly for literals to ensure that paddings are zeroed.                                                |

**Operand Key:**

- `Rd`: Destination Register (8-bit register index)
- `Rn`: Source Register n (8-bit register index)
- `Rm`: Source Register m (8-bit register index)
- `Rp`: Source Register p (8-bit register index)
- `#imm8`: 8-bit immediate value
- `#imm32`: 32-bit immediate value
- `#offset`: Offset value (can be positive or negative, size dependent on
  instruction encoding)
- `#fp_offset`: Frame Pointer unsigned Offset value.
- `#branch_target`. Offset for the PC.
- `#branch_absolute`. Absolute u32 instruction offset to call into.
- `#absolute_addr`: Absolute unsigned u32 memory address.
- `#size`: Size value (size dependent on instruction encoding)
- `#len`: Length value (size dependent on instruction encoding)
- `[...]`: Memory address calculated from base register and offset.

**Internal registers:**

- SP. Stack Pointer. Modified by `enter` and `ret` opcode.
- FP. Frame pointer. Points to the "start" of the allocated stack frame.
- PC. Program Counter, holds the offset to the instruction
- Flags. Only Z flag is supported.
- R0-R255. General purpose 32-bit registers.

**Fixed point:**

It is a signed 32 bit value with 16 bits for integer part and 16 bits for the floating part. (16.16).

**Instruction execution:**

- Instruction is fetched from PC.
- PC is incremented (PC += 1).
- The instruction is executed.
